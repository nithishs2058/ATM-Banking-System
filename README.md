🏦 SmartBank ATM System – Python Project

A console-based ATM banking system built using Python. Users can authenticate with a PIN, check their account balance, deposit money, withdraw funds, and view transaction history.

📌 About The Project

This project is a beginner-friendly Python application designed to demonstrate core programming concepts:

User input handling

Conditional statements

Loops

Functions

Lists and dynamic data storage

The system runs in the terminal and allows users to perform multiple banking operations until they choose to exit.

🎯 Features

PIN Authentication – Secure access with maximum 3 attempts

Check Balance – View the current account balance

Deposit Money – Add funds to your account

Withdraw Money – Withdraw funds if sufficient balance is available

Transaction History – Track all deposits and withdrawals

Exit Option – Close the application safely

Input validation and clear console messages

🛠️ Technologies Used

Python 3

Console / Terminal Interface

▶️ How to Run

Make sure Python 3 is installed on your system.

Download or clone this repository:

git clone https://github.com/your-username/atm-python-project.git

Navigate to the project folder:

cd atm-python-project

Run the program:

python atm_system.py
